# Reel 16 — Building a Wix Site in 60 Seconds (Day 16)

**Duration:** 45–60 seconds
**Format:** Vertical 9:16 (1080×1920)
**Recording method:** Screen recording of a Wix Studio build session, sped up
**Voiceover:** None

---

## Why this works

People love timelapses. "How is it made" content converts browsers into clients because it shows competence without bragging. This is the single most important Reel in the client-acquisition arc.

---

## How to produce

### Option A — Record a real build

If you have an active or upcoming Wix Studio project:

1. Open Wix Studio in Chrome, fullscreen (F11)
2. Start a screen recording BEFORE you begin working
3. Build 15–30 minutes of actual site work (layout, styling, adding content)
4. Stop recording
5. Import to CapCut, speed up 10–20x to compress into 45–60 seconds
6. Cut out any pauses, mouse wandering, or screens that show client-specific info

### Option B — Recreate on a demo

If no active project:

1. Start a new Wix Studio template (pick a clean starter)
2. Spend 15 minutes transforming it — replace copy, adjust colors, rearrange blocks
3. Same recording + speed-up process

**Important:** Don't show any real client data, real emails, or anything under NDA. If in doubt, use placeholder Lorem Ipsum content.

---

## Shot sequence

### Opener (0:00–0:03)
- Black screen with text overlay: **"From blank → built."**
- Dissolve into your Wix editor view

### Main body (0:03–0:50)
- Sped-up editor footage (10–20x)
- Every 10 seconds of real time, add a small milestone text overlay:
  - "01 / LAYOUT"
  - "02 / BRAND COLORS"
  - "03 / HERO SECTION"
  - "04 / SERVICES"
  - "05 / CONTACT"
  - "06 / MOBILE TUNE"

### Reveal (0:50–0:55)
- Slow down to real time
- Preview the finished page
- Scroll through the final site at normal speed

### CTA (0:55–1:00)
- Fade to black
- Text: **"Wix Studio subscription."**
- Text: **"From $149/mo. Cancel anytime."**
- URL: **techturi.org/book**

---

## Text overlay style

- JetBrains Mono, 60pt minimum
- Bone color with chartreuse accent words
- Position alternating top-third and bottom-third
- Simple fade in/out animations
