# Reel 11 — $0 Certifications That Got Me Hired (Day 11)

**Duration:** 25–35 seconds
**Format:** Vertical 9:16 (1080×1920)
**Recording method:** Screen-record the `reel-11-free-certs.html` animation in this folder, then add music in CapCut
**Voiceover:** None

---

## Why this format

Typography-only Reels convert incredibly well for "list" content. No video skills required. You literally screen-record an HTML file playing in your browser.

---

## How to produce this

1. Open `reel-11-free-certs.html` in Chrome (fullscreen, F11)
2. Start a screen recording of the browser window
3. Refresh the page — the animation runs automatically
4. Let it run once fully (~30 seconds)
5. Stop recording
6. In CapCut: crop to 9:16 vertical, add music, export 1080×1920

---

## The HTML animation file

The companion file `reel-11-free-certs.html` in this folder contains a ready-to-record animated typography sequence. Open it in Chrome, go fullscreen, refresh, and screen-record.

Content sequence:
